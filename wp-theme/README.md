# Tema WordPress - Dra. Joseane Souza

Tema profissional para WordPress desenvolvido para Dra. Joseane Souza, Neuropsicoterapeuta especializada em ansiedade e traumas emocionais.

## 📁 Estrutura do Tema

```
dra-joseane-souza/
├── assets/
│   ├── images/
│   │   └── dra-joseane.png
│   └── js/
│       └── main.js
├── style.css              # Estilos principais + metadados do tema
├── functions.php          # Funções e configurações do tema
├── header.php             # Cabeçalho do site
├── footer.php             # Rodapé do site
├── front-page.php         # Template da página inicial
├── index.php              # Template padrão para posts
├── page.php               # Template para páginas
└── screenshot.png         # Screenshot do tema (1200x900px)
```

## 🚀 Instalação

1. Baixe a pasta `dra-joseane-souza`
2. Acesse o painel do WordPress > Aparência > Temas > Adicionar Novo
3. Clique em "Enviar tema" e selecione o arquivo ZIP da pasta
4. Ative o tema

## ⚙️ Configurações

### Customizer

Acesse **Aparência > Personalizar** para configurar:

#### Opções do Tema
- Número do WhatsApp
- Mensagem padrão do WhatsApp
- Email de contato
- Telefone
- Localização

#### Redes Sociais
- Instagram URL
- LinkedIn URL

### Custom Post Types

O tema inclui dois tipos de post personalizados:

#### Depoimentos
- Gerencie depoimentos de pacientes
- Campos: Título (nome), Conteúdo (depoimento), Localização, Avaliação (1-5 estrelas)

#### FAQ
- Gerencie perguntas frequentes
- Campos: Título (pergunta), Conteúdo (resposta)

## 🎨 Design System


### Tipografia

- **Títulos:** Playfair Display (serif)
- **Corpo:** Inter (sans-serif)

### Animações

- `animate-fade-in` - Fade in com movimento vertical
- `animate-fade-in-left` - Fade in da esquerda
- `animate-fade-in-right` - Fade in da direita
- `animate-float` - Flutuação suave
